超详细创建Django项目并存在Gitee仓库中

1.创建全局豆瓣镜像(提高下载速度)

windows环境下:

写入
[global]
index-url=https://pypi.doubanio.com/simple

linux环境下:

// 写入
[global]
index-url=https://pypi.doubanio.com/simple

2.安装django

3.创建项目

4.用Pycharm远程clone项目

4.1 克隆项目

在Pycharm中git下仓库(url:git@gitee.com:wang_ba_tian/djangocase.git)

4.2 创建虚拟环境

在file下的setting创建虚拟环境

点击选择add

就用默认的数据,直接点ok

创建成功后的样子如下

4.3 下载安装依赖项

创建成功后在Pycharm左下方的Termial中下载依赖项

(venv) E:\PythonProject\djangocase>pip install -r requirements.txt

更新requirements.txt中的依赖项
(venv) E:\PythonProject\djangocase>pip freeze > requirements.txt

4.4 添加配置

在下载的同时,添加配置

配置完成后点ok,此时依赖环境也应该安装完成了,可以在even中查看这些东西

4.5 修改文件配置

修改项目下与项目文件同名的文件夹下的settings.py文件内容

…

TEMPLATES = [
{
‘BACKEND’: ‘django.template.backends.django.DjangoTemplates’,
// 配置模板路径
‘DIRS’: [os.path.join(BASE_DIR, ‘templates’)],
….
},
]
…
//配置数据库
DATABASES = {
‘default’: {
// 数据库类型
‘ENGINE’: ‘django.db.backends.mysql’,
// 数据库名字
‘NAME’: ‘djangocase’,
// 数据库地址
‘HOST’: ‘本地或者服务器地址’,
// 端口
‘POST’: 3306,
// 数据库用户名
‘USER’: ‘用户名’,
// 密码
‘PASSWORD’: ‘密码’,
// 编码方式
‘CHARSET’: ‘utf8’,
// 设置时区
‘TIME_ZONE’: ‘Asia/Chongqing’
}
}
…
//配置时区
TIME_ZONE = ‘Asia/Chongqing’
…
// 配置静态资源路径
STATICFILES_DIRS = [os.path.join(BASE_DIR, ‘static’)]

STATIC_URL = ‘/static/‘

至此,项目创建完成,可以开始你的项目编写了.

超详细创建Django项目并存在Gitee仓库中

已做免密是配置,复制仓库ssh的URL,若未做免密配置,请选择https的URL

本文是已做免密设置,与未作免密配置的差别就在于连接仓库的时候是否验证而已

1.创建全局豆瓣镜像(提高下载速度)

windows环境下:

Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ cd

Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ mkdir pip

Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ cd pip

Administrator@DESKTOP-VJ0GDUL MINGW64 ~/pip
$ vim pip.ini
写
[global]
index-url=https://pypi.doubanio.com/simple

linux环境下:
Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ cd

Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ mkdir .pip

Administrator@DESKTOP-VJ0GDUL MINGW64 ~
$ cd .pip

Administrator@DESKTOP-VJ0GDUL MINGW64 ~/pip
$ vim pip.conf
写
[global]
index-url=https://pypi.doubanio.com/simple

3.创建项目
// 创建django项目
Administrator@DESKTOP-VJ0GDUL MINGW64 /d
$ django-admin startproject djangocase

Administrator@DESKTOP-VJ0GDUL MINGW64 /d
$ cd djangocase/

// 创建仓库
Administrator@DESKTOP-VJ0GDUL MINGW64 /d/djangocase
$ git init
Initialized empty Git repository in D:/djangocase/.git/

// 设置忽略文件(写入vevn,.idea和http://gitignore.io/这个网址生成的(windows,macOS,Linux,python,Pycharm))
Administrator@DESKTOP-VJ0GDUL MINGW64 /d/djangocase (master)
$ vim .gitignore

// 写入依赖项
// Django==2.2.9,django-redis,mysqlclient等
Administrator@DESKTOP-VJ0GDUL MINGW64 /d/djangocase (master)
$ vim requirements.txt

// 将文件添加到缓存区
Administrator@DESKTOP-VJ0GDUL MINGW64 /d/djangocase (master)
$ git add .

4.用Pycharm远程clone项目

在Pycharm中git下仓库(url:git@gitee.com:wang_ba_tian/djangocase.git)

在file下的setting创建虚拟环境

点击选择add

创建成功后在Pycharm下方的Termial中下载依赖项

(venv) E:\PythonProject\djangocase>pip install -r requirements.txt
更新requirements.txt中的依赖项
(venv) E:\PythonProject\djangocase>pip freeze > requirements.txt

修改项目下的urls

…

TEMPLATES = [
{
‘BACKEND’: ‘django.template.backends.django.DjangoTemplates’,
// 配置模板路径
‘DIRS’: [os.path.join(BASE_DIR, ‘templates’)],
….
},
]
…
//配置数据库
DATABASES = {
‘default’: {
// 数据库类型
‘ENGINE’: ‘django.db.backends.mysql’,
// 数据库名字
‘NAME’: ‘djangocase’,
// 数据库地址
‘HOST’: ‘本地或者服务器地址’,
// 端口
‘POST’: 3306,
// 数据库用户名
‘USER’: ‘用户名’,
// 密码
‘PASSWORD’: ‘密码’,
// 编码方式
‘CHARSET’: ‘utf8’,
// 设置时区
‘TIME_ZONE’: ‘Asia/Chongqing’
}
}
…
//配置时区
TIME_ZONE = ‘Asia/Chongqing’
…
// 配置静态资源路径
STATICFILES_DIRS = [os.path.join(BASE_DIR, ‘static’)]

STATIC_URL = ‘/static/‘

至此,项目创建完成,可以开始你的项目编写了.

题外话: 工作时,项目一般情况都是直接git clone公司的,不用自己去创建.
